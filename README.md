
📌 Project Overview

This project introduces essential Pandas operations through real-world movie data examples.

You will learn how to:

Create DataFrames
Explore datasets
Filter and query data
Sort and aggregate information
Handle missing values
Perform basic statistical analysis
Visualize data using Matplotlib

🛠 Technologies Used
Python 3
Pandas
Matplotlib

Install required libraries: pip install pandas matplotlib

📈 Visualizations Included

Genre Distribution
Shows the number of movies in each genre.

Rating vs Release Year
Displays movie ratings over time.

Rating Histogram
Shows the distribution of ratings.

▶️ Running the Project

Run the Python script: python movie_analysis.py

📚 Learning Outcomes

After completing this project, you will understand:

Core Pandas DataFrame operations
Data cleaning basics
Aggregation and grouping
Statistical analysis
Simple data visualization techniques

🎯Goal

Practice fundamental data analysis skills using Pandas.
